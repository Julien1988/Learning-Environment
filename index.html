What'is MarkDown ?

  • C'est un éditeur de texte
  • Il propose une synthaxe clair et efficace, de manière à pouvoir laisser des explications sur le code, les repos GitHub etc..
  • Il permet aussi de pouvoir y ajouter des lignes de code comme celle-ci :
<script type="text/javascript">
    alert("Hello!");
</script>
